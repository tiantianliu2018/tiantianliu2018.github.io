JavaBean

JavaBean 是一种 Java 语言写成的可重用组件。

所谓JavaBean，是指符合如下标准的Java类：

  • 类是公共的

  • 有一个无参的公共构造器

  • 有属性，属性一般是私有的，且有对应的get、set方法
