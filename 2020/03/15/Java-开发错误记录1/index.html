!error

以上代码看似没有问题，往 list 中添加 FeedContent 对象，但最后结果全是重复数据！

为什么呢？

因为将对象 add 进 list 时，放入 list 的是对象的引用，虽然每次为对象重新 set 值，但实际上仍然是同一个对象，每次都是重新赋值覆盖之前的数据，所以循环的都是重复的数据。

因此 new 对象应该放在循环里面，保证每次加入 list 的都是新的对象。
